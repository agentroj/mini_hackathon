Projects

Here are some of the most significant software projects I’ve designed and developed — spanning backend systems, AI automation, and large-scale data integrations. Each project demonstrates my experience in architecture, API design, and scalable deployment.

TicketMonster

An AI-powered event ticketing platform built with Django REST Framework and React. Features predictive attendance analytics, smart QR check-ins, affiliate systems, and timed entry slot management. Deployed on GCP with CI/CD via GitHub Actions.

  • Stack: Django, React, Redis, Celery, PostgreSQL, Docker
  • Highlights: Predictive models, real-time analytics dashboard

ChoirConnect

A social platform for choirs and music organizations featuring profile management, booking systems, content sharing, and an internal marketplace for digital sheet music.

  • Stack: Django + React + Supabase + Docker
  • Highlights: Role-based access, cloud media uploads, API authentication

RhythmiCoach

An interactive rhythm-training app for musicians with real-time feedback and progress tracking. Uses Web Audio API for beat detection and Django backend for analytics.

  • Stack: Django, React, Web Audio API, PostgreSQL
  • Highlights: Real-time feedback, gamified progress tracking

Magnet_X (AI Document Assistant)

A LangChain-powered platform for PDF extraction and contextual AI chat. Integrates FAISS for vector storage and supports retrieval-augmented generation (RAG) pipelines.

  • Stack: FastAPI, LangChain, FAISS, Supabase, Docker
  • Highlights: Embedding pipelines, query logging, and analytics dashboard

ETL & Data Integration Suite

Custom Django + Celery data-pipeline system for automating Shopify, CheckoutChamp, and SFTP integrations with BigQuery. Handles 50,000+ records daily with detailed logging and retry mechanisms.

  • Stack: Django, Celery, Redis, BigQuery, PostgreSQL
  • Highlights: Automated ETL jobs, audit logging, dynamic schema mapping

AI Forex Assistant

A full-stack web app providing AI-driven forex trading signals, insights, and recommendation dashboards using OpenAI GPT APIs and Django backend.

  • Stack: Django, React, OpenAI API, PostgreSQL
  • Highlights: Automated insights, trend analysis, secure user dashboard

Each project reflects my focus on performance, scalability, and clean architecture. I’m continuously exploring new ways to integrate AI, automation, and data pipelines into real-world systems.
